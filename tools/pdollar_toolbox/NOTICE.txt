Piotr Dollár MATLAB Toolbox

This directory contains code from the Piotr Dollár MATLAB Toolbox,
which is required by the official BSDS500 evaluation benchmark.

Author:
  Piotr Dollár

Usage:
  This toolbox is used as a dependency of the BSDSBench evaluation code
  (e.g., fevalDistr and related utilities).
