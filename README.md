# DMOR-Edge

Dynamic Modulated Operator Router for Lightweight Edge Detection

## Overview
DMOR-Edge is a lightweight edge detection framework that dynamically routes
multiple complementary edge operators via a modulated operator router.

Unlike fixed multi-branch architectures, DMOR-Edge performs input-aware and
spatial-aware operator selection to enhance boundary localization while
suppressing texture noise.

## Key Contributions
- Operator-space edge representation with complementary inductive biases
- Dynamic modulated operator routing (global + spatial)
- Top-K sparse operator selection for efficiency and interpretability
- Lightweight design suitable for real-time edge detection
